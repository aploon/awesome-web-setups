# Next.js Starter Setup

Un setup minimaliste pour démarrer avec Next.js.

- ⚡️ Next.js 15 avec App Router
- 💎 TypeScript pour un code plus robuste

- 📱 Layout responsive par défaut
- 🌐 Optimisations SEO de base
